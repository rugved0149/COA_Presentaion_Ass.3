Macros Definition

WRITE Macro

%macro WRITE 2
    mov eax, 4          ; sys_write
    mov ebx, 1          ; fd = stdout
    mov ecx, %1         ; buffer
    mov edx, %2         ; length
    int 0x80
%endmacro

Purpose: Output text to stdout using Linux sys_write (syscall #4)

READ Macro

%macro READ 2
    mov eax, 3          ; sys_read
    mov ebx, 0          ; fd = stdin
    mov ecx, %1         ; buffer
    mov edx, %2         ; length
    int 0x80
%endmacro

Purpose: Read input from stdin using Linux sys_read (syscall #3)

EXIT Macro

accept Function - Hex String Parsing

accept:
    xor ebx, ebx        ; clear result
    mov esi, buf        ; source pointer
.loop:
    mov al, [esi]       ; get character
    cmp al, 10          ; newline?
    je .done
    cmp al, 0           ; null terminator?
    je .done
    ; Check if digit 0-9
    cmp al, '0'
    jb .done
    cmp al, '9'
    jbe .digit
    ; Check if uppercase A-F
    cmp al, 'A'
    jb .chk_low
    cmp al, 'F'
    jbe .up
.chk_low:
    ; Check if lowercase a-f
    cmp al, 'a'
    jb .done
    cmp al, 'f'
    ja .done
    sub al, 'a'-10      ; convert to number
    jmp .got
.up:
    sub al, 'A'-10      ; convert to number
    jmp .got
.digit:
    sub al, '0'         ; convert to number
.got:
    shl ebx, 4          ; shift left 4 bits
    add ebx, eax        ; add new nibble
    inc esi             ; next character
    jmp .loop
.done:
    ret                 ; result in EBX

Parsing Algorithm

  1. Initialize result (EBX) to zero
  2. Read each character from input buffer
  3. Check if valid hex digit (0-9, A-F, a-f)
  4. Convert character to numeric value (0-15)
  5. Shift previous result left by 4 bits
  6. Add new nibble to result
  7. Repeat until end of string

display_hex8 Function - Hex Output

display_hex8:
    mov esi, buf        ; output buffer
    mov ecx, 8          ; 8 hex digits
.next:
    rol ebx, 4          ; rotate left 4 bits
    mov al, bl          ; get low byte
    and al, 0x0F        ; mask to get nibble
    cmp al, 9           ; check if > 9
    jbe .num
    add al, 7           ; adjust for A-F
.num:
    add al, '0'         ; convert to ASCII
    mov [esi], al       ; store character
    inc esi             ; next position
    loop .next          ; repeat 8 times
    WRITE buf, 8        ; output 8 characters
    ret

Output Process

1. Extract Nibble

Rotate EBX left 4 bits, mask to get 4-bit value

2. Convert to ASCII

0-9 → '0'-'9', 10-15 → 'A'-'F'

3. Store Character

Place ASCII character in output buffer

4. Repeat

Process all 8 nibbles (32 bits)

Example: 0x1A2B3C4D → "1A2B3C4D"

Linux System Calls Interface

System Call Numbers & Registers

Syscall Number EAX EBX ECX EDX
sys_exit 1 1 exit_code - -
sys_read 3 3 fd (0=stdin) buffer count
sys_write 4 4 fd (1=stdout) buffer count

System Call Mechanism

1. Setup Registers

Load syscall number in EAX, parameters in EBX, ECX, EDX

2. Software Interrupt

Execute int 0x80 to switch to kernel mode

3. Kernel Processing

Linux kernel handles the system call request

4. Return to User

Control returns to user program with result in EAX
